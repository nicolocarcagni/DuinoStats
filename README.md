# DuinoStats
Your Duino dashboard in your Terminal.

## Installing
Before using it, you have to install [pyduinocoin](https://github.com/BackrndSource/pyduinocoin)

```bash
$ pip install pyduinocoin
```

## Execute

```bash
python duinostats.py
```
